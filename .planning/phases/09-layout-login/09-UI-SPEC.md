## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage in this phase |
|-------|-------|---------------------|
| xs | 4px (p-1) | Icon gaps inside sidebar menu buttons, tooltip internal padding |
| sm | 8px (p-2) | Sidebar menu item vertical padding, gap between nav items, form field internal spacing |

## Interaction Contracts

### Sidebar Collapse/Expand

| Interaction | Behavior |
|-------------|----------|
| Click SidebarRail | Toggle between expanded (256px) and collapsed (48px). Animated via CSS transition `width 200ms ease-linear`. |
| Click SidebarTrigger | Same as SidebarRail click. Trigger button is in the page header area (inside SidebarInset). |
| Press Cmd+B (Mac) / Ctrl+B (Win) | Same as SidebarRail click. Built into shadcn SidebarProvider. |
| Sidebar transition completes | Fire `transitionend` listener. Call `map.invalidateSize()` for Leaflet maps. Dispatch `window.dispatchEvent(new Event('resize'))` for Recharts charts. |
| Cookie persistence | On toggle: set cookie `sidebar_state=expanded` or `sidebar_state=collapsed`, `max-age=604800` (7 days), `path=/`. |

### Sidebar Navigation

| Interaction | Behavior |
|-------------|----------|
| Click nav item (expanded) | Navigate to route. Active item shows `var(--sidebar-accent)` background + `var(--sidebar-accent-foreground)` text. |
| Click nav item (collapsed) | Same navigation. Item is icon-only. |
| Hover nav item (collapsed) | Show Tooltip to the right of icon with menu item label. Tooltip delay: 0ms (instant). |
| Hover nav item (expanded) | Subtle background highlight `bg-sidebar-accent/50`. |
| Active state detection | Match current `pathname` against item `href`. Dashboard uses exact match; all others use prefix match. |

### Sidebar Mobile

| Interaction | Behavior |
|-------------|----------|
| Screen < 768px (md breakpoint) | Sidebar renders inside Sheet component (overlay). No SidebarRail. |
| Click hamburger menu | Opens Sheet from left side. Full sidebar content visible. |
| Click nav item (mobile) | Navigate + auto-close Sheet. |
| Click outside Sheet | Close Sheet. |
| Swipe left on Sheet | Close Sheet (built into Sheet component). |

### Sidebar Footer

| Interaction | Behavior |
|-------------|----------|
| Expanded state | Shows Avatar (32x32) + User name (`text-sm font-semibold`) + email (`text-xs text-muted-foreground`). Non-interactive display. |
| Collapsed state | Shows Avatar only (32x32), centered. Click opens DropdownMenu to the right. |
| Click avatar (collapsed) | Opens DropdownMenu with user info + sign out action. |
| Click "Sign out" | Calls `authClient.signOut()`, redirects to `/sign-in`. |

### Login Form

| Interaction | Behavior |
|-------------|----------|
| Submit form | Validate with Zod schema. If valid, call `authClient.signIn.email()` with `rememberMe` option based on checkbox state. Show Loader2 spinner on button during request. |
| Remember me checked | Pass `rememberMe: true` to `signIn.email()` -- session extends to 30 days. |
| Remember me unchecked | Pass `rememberMe: false` to `signIn.email()` -- session expires when browser closes. |
| Validation error (client) | Show inline `text-xs text-destructive` below the offending field. |
| Auth error (server) | Show `text-sm text-destructive` centered message above Sign In button: "Invalid email or password. Please try again." |
| Successful login (admin) | Redirect to `/admin`. |
| Successful login (tenant) | Auto-set active org if single membership, redirect to `/app/dashboard`. |
| Loading state | Button disabled, shows `Loader2` spinning icon. Both inputs disabled. |

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| Sidebar toggle | `SidebarTrigger` has `aria-label="Toggle Sidebar"`, reflects state via `data-state="expanded"` or `data-state="collapsed"` |
| Sidebar nav | `<nav>` element with `aria-label="Main Navigation"` wrapping `SidebarContent` |
| Active nav item | `aria-current="page"` on the active `SidebarMenuButton` link |
| Collapsed tooltips | Tooltip provides accessible name for icon-only buttons via `aria-label` passthrough |
| Keyboard shortcut | Cmd+B / Ctrl+B announced nowhere (discoverable shortcut, not required for accessibility) |
| Mobile sidebar | Sheet has `role="dialog"`, `aria-label="Navigation"`, focus trapped inside while open |
| Login form | `<form>` with `aria-label="Sign in"`, fields linked to labels via `htmlFor`/`id` |
| Login errors | Error messages use `aria-live="polite"` for screen reader announcement |
| Remember me | Checkbox has explicit `<Label>` association, `aria-checked` managed by component |
| Branding panel | Decorative only -- `aria-hidden="true"` on decorative dot pattern |
| Skip nav | Existing skip-to-content link (if present) continues to work with `SidebarInset` as main content |

| Element | Duration | Easing | Property |
|---------|----------|--------|----------|
| Sidebar width change | 200ms | ease-linear | `width` |
| Sidebar content opacity | 200ms | ease-linear | `opacity` (labels fade in/out during collapse) |
| Mobile Sheet open/close | 300ms | ease-in-out | `transform` (slide from left) |
| Login button loading | Instant | -- | Swap text for Loader2 spinner |
| Nav item hover highlight | 150ms | ease-in-out | `background-color` |
| Tooltip appear | 0ms delay | -- | Immediate show on hover (collapsed sidebar needs instant feedback) |
