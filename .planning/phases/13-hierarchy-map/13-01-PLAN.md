| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-13-02 | Tampering | Tree node ID in API calls | accept | Node IDs are UUIDs from trusted API responses, passed as URL path params. Backend validates ownership via RLS. No user-supplied free-text IDs. |
