     ```typescript
     const rowActions: RowAction<WebhookRow>[] = [

       { label: row.original.isActive ? "Disable" : "Enable", icon: ToggleLeft, onClick: callbacks.onToggle },

       { label: "Delete", icon: Trash2, onClick: callbacks.onDelete, variant: "destructive" },
     ]
     ```
     Since DataTableRowActions receives static actions per row, and we need per-row dynamic label ("Disable"/"Enable"), the actions column cell function must build the array dynamically for each row. This means the actions column definition gets the callbacks object via closure and constructs RowAction[] inline.
