Key methods:

- `async ensureBucket(orgId: string)`: Creates bucket `org-${orgId}` if not exists.
- `async uploadSegment(orgId: string, objectPath: string, buffer: Buffer, size: number)`: Upload buffer to `org-${orgId}` bucket at `objectPath`. Use `this.client.putObject(bucket, objectPath, buffer, size)`.
- `async getPresignedUrl(orgId: string, objectPath: string, expirySeconds = 14400)`: Returns pre-signed GET URL. 4-hour default expiry to avoid mid-playback expiration (per Pitfall 6).
- `async removeObject(orgId: string, objectPath: string)`: Delete object.
- `async removeObjects(orgId: string, objectPaths: string[])`: Bulk delete using `this.client.removeObjects(bucket, objectPaths)`.
- `async getObjectStream(orgId: string, objectPath: string)`: Returns readable stream for backend proxy playback.

**4. recordings.service.ts:**
Injectable service with constructor injecting `TENANCY_CLIENT` (Prisma), `MinioService`, `PrismaService` (raw, for cross-org queries like Package lookup).

Key methods:

- `async startRecording(cameraId: string, orgId: string): Promise<Recording>`:
  1. Check camera exists and is online (query via tenancy client)
  2. Check camera is not already recording (`isRecording === true` rejects)
  3. Check storage quota via `checkStorageQuota(orgId)` -- reject if at 100%
  4. Create `Recording` record with `status: 'recording'`, `orgId`, `cameraId`
  5. Update camera `isRecording = true`
  6. Ensure MinIO bucket exists via `minioService.ensureBucket(orgId)`
  7. Return the created Recording

- `async stopRecording(cameraId: string, orgId: string): Promise<Recording>`:
  1. Find active recording for camera (`status: 'recording'`)
  2. Update Recording: `status: 'complete'`, `stoppedAt: new Date()`
  3. Update camera `isRecording = false`
  4. Return updated Recording

- `async getActiveRecording(cameraId: string, orgId: string): Promise<Recording | null>`:
  Query via tenancy client: `recording.findFirst({ where: { cameraId, orgId, status: 'recording' } })`

- `async archiveSegment(recordingId: string, orgId: string, cameraId: string, data: { filePath: string, duration: number, seqNo: number, url: string, m3u8Path: string })`:
  1. **Path validation (T-07-01):** Validate `filePath` starts with allowed prefix (configurable via `SRS_HLS_PATH` env var, default `/srs-hls`). Reject paths containing `..` to prevent path traversal.
  2. Read segment file from `filePath` using `fs.readFile()` (not stream, segment is small ~50KB for 2s fragment)
  3. Get file size via buffer.length
  4. Check if this is the first segment for this recording (no existing segments) -- if so, parse the m3u8 file at `data.m3u8Path` to find `#EXT-X-MAP:URI="..."` and archive the init segment. Store init segment path in Recording.initSegment.
  5. Generate object path per D-04: `${cameraId}/${YYYY-MM-DD}/${HH-MM-SS}_${seqNo}.m4s` using current timestamp
  6. Upload to MinIO via `minioService.uploadSegment(orgId, objectPath, buffer, buffer.length)`
  7. Create `RecordingSegment` record with `orgId, recordingId, cameraId, objectPath, duration, size: BigInt(buffer.length), seqNo, timestamp: new Date()`
  8. Update Recording `totalSize += BigInt(buffer.length)`, `totalDuration += duration`

- `async checkStorageQuota(orgId: string): Promise<{ allowed: boolean, usageBytes: bigint, limitBytes: bigint, usagePercent: number }>`:
  1. Get org's package via raw PrismaService: `organization.findUnique({ where: { id: orgId }, include: { package: true } })`
  2. Get total storage used: `recordingSegment.aggregate({ where: { orgId }, _sum: { size: true } })`
  3. Calculate `limitBytes = BigInt(package.maxStorageGb) * BigInt(1024 * 1024 * 1024)`
  4. Calculate `usagePercent = Number(usageBytes * 100n / limitBytes)`
  5. Return `{ allowed: usagePercent < 100, usageBytes, limitBytes, usagePercent }`

- `async listRecordings(cameraId: string, orgId: string, date?: string)`: Query recordings for camera, optionally filtered by date.

- `async getRecording(id: string, orgId: string)`: Get single recording with segment count.

```typescript
@Post('on-hls')
async onHls(@Body() body: any) {
  const parsed = onHlsCallbackSchema.safeParse(body);
  if (!parsed.success) {
    this.logger.warn(`Invalid on_hls callback: ${JSON.stringify(parsed.error.issues)}`);
    return { code: 0 };
  }

  const { orgId, cameraId } = this.parseStreamKey(parsed.data.stream, parsed.data.app);
  if (!orgId || !cameraId) {
    return { code: 0 }; // Internal stream, skip
  }

  try {
    const recording = await this.recordingsService.getActiveRecording(cameraId, orgId);
    if (!recording) {
      return { code: 0 }; // Not recording, skip
    }

    const quota = await this.recordingsService.checkStorageQuota(orgId);
    if (!quota.allowed) {
      this.logger.warn(`Storage quota exceeded for org=${orgId}, skipping archive`);
      return { code: 0 };
    }

    // Resolve file path for the API container
    // SRS sends relative path like ./objs/nginx/html/live/...
    // Map to the mount point the API container uses
    const hlsMountPath = process.env.SRS_HLS_PATH || '/srs-hls';
    const segmentFile = parsed.data.file.replace(/^\.\/objs\/nginx\/html/, hlsMountPath);
    const m3u8File = parsed.data.m3u8.replace(/^\.\/objs\/nginx\/html/, hlsMountPath);

    await this.recordingsService.archiveSegment(recording.id, orgId, cameraId, {
      filePath: segmentFile,
      duration: parsed.data.duration,
      seqNo: parsed.data.seq_no,
      url: parsed.data.url,
      m3u8Path: m3u8File,
    });
  } catch (err) {
    // Fire-and-forget pattern: log error but don't block SRS
    this.logger.error(`Failed to archive segment: ${err.message}`, err.stack);
  }

  return { code: 0 };
}
```
