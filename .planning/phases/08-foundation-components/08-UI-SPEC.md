# Phase 8 — UI Design Contract

> Visual and interaction contract for the reusable DataTable system and DatePicker/DateRangePicker components. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn (base-nova style) |
| Preset | base-nova (neutral base, CSS variables, Tailwind 4.2) |
| Component library | base-ui (render prop pattern, NOT Radix asChild) |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage in this phase |
|-------|-------|---------------------|
| xs | 4px (p-1) | Gap between checkbox and cell content, icon internal padding |
| sm | 8px (p-2) | Table cell padding (existing `TableCell` uses `p-2`), toolbar item gaps |
| md | 16px (p-4) | Toolbar padding, space between toolbar and table, space between table and pagination |
| lg | 24px (p-6) | Card/container internal padding for empty states |
| xl | 32px (p-8) | Page-level content padding (existing `main` uses `p-4 md:p-8`) |

Exceptions: Table header height is 40px (`h-10`, existing `TableHead` pattern). Minimum touch target for row action button is 28px (`size-7`, existing `icon-sm` button size).

## Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body / Table cell | 14px | 400 (normal) | 1.43 (20px) | `text-sm` |
| Label / Table header | 14px | 500 (medium) | 1.43 (20px) | `text-sm font-medium` |
| Heading (page title) | 20px | 600 (semibold) | 1.4 (28px) | `text-xl font-semibold` |
| Caption / Muted text | 12px | 400 (normal) | 1.33 (16px) | `text-xs` |

Source: Existing `table.tsx` uses `text-sm` for body. Existing `TableHead` uses `font-medium`. Badge uses `text-xs font-medium`. Button uses `text-sm font-medium`.

Weights used in this phase: 400 (normal) and 500 (medium). 600 (semibold) reserved for page headings only.

## Color

| Role | Value | Usage in this phase |
|------|-------|---------------------|
| Dominant (60%) | `hsl(0 0% 100%)` / `var(--background)` | Page background, table background |
| Secondary (30%) | `hsl(220 14% 96%)` / `var(--muted)` | Table row hover (`bg-muted/50`), selected row (`bg-muted`), toolbar background, pagination area |
| Accent (10%) | `hsl(142 71% 45%)` / `var(--primary)` | Primary action buttons ("Add Camera", "Add API Key"), active sort indicator arrow, checkbox checked state, focus ring |
| Destructive | `hsl(0 84% 60%)` / `var(--destructive)` | Delete menu item text, bulk delete button, destructive confirmation dialog |

Accent reserved for:
- Primary CTA buttons in toolbar (e.g., "Add Camera")
- Checkbox checked fill color
- Active column sort direction arrow
- Focus ring on interactive elements (`--ring`)
- Pagination active page indicator

NOT used for: table borders, filter chips at rest, row action "..." button, pagination arrows.

## Interaction Contracts

### DataTable

| Interaction | Behavior |
|-------------|----------|
| Column header click | Toggles sort: none -> ascending -> descending -> none. Arrow icon updates to reflect state. |
| Search input | Filters rows by global text match. Debounced at 300ms. Clears with X button. |
| Faceted filter chip click | Opens Popover below chip. Multi-select checkboxes inside. Chip shows count badge when filters active (e.g., "Status (2)"). |
| Faceted filter "Clear filters" | Resets all faceted filters. Button appears only when filters are active. |
| Row checkbox | Toggles row selection. Header checkbox toggles select-all-on-current-page. |
| Row hover | `bg-muted/50` background (existing `TableRow` pattern). |
| Row selected | `bg-muted` background (existing `data-[state=selected]:bg-muted` pattern). |
| "..." row action click | Opens DropdownMenu aligned to end. Items defined per-page via column definition. |
| Pagination page click | Switches to target page. Current page uses accent background. |
| Pagination "Previous"/"Next" | Disabled at boundaries (first/last page). Uses `Button variant="outline" size="sm"`. |
| Page size selector | Select dropdown with options: 10, 25, 50. Default: 10. |
| Empty state (no data) | Centered icon + heading + description + optional CTA button. Full table-width, replaces table body. |
| Empty state (filtered, no results) | Centered message "No results found" + "Try adjusting your filters" + "Clear filters" button. |
| Loading state | Skeleton rows matching column count. 5 skeleton rows with pulse animation. Uses existing `Skeleton` component. |

### DatePicker

| Interaction | Behavior |
|-------------|----------|
| Button click | Opens Popover with Calendar below button, aligned to start. |
| Date selection | Selects date, closes Popover, button label updates to formatted date (e.g., "Apr 17, 2026"). |
| No date selected | Button shows placeholder text in muted-foreground color (e.g., "Pick a date"). |
| CalendarIcon | 16x16 icon at left side of button label, `text-muted-foreground`. |

### DateRangePicker

| Interaction | Behavior |
|-------------|----------|
| Button click | Opens Popover with Calendar (range mode) below button. |
| First click | Selects range start. Calendar highlights start date. |
| Second click | Selects range end. Calendar highlights full range. Popover closes. Button updates to "Apr 10 - Apr 17, 2026". |
| No range selected | Button shows "Pick a date range" in muted-foreground. |
| CalendarIcon | 16x16 icon at left side of button label, `text-muted-foreground`. |

## Layout Specifications

### DataTable Toolbar

```
[Search Input (240px)]  [Faceted Filter Chip] [Faceted Filter Chip]  ----stretches----  [Action Button]
```

- Search input: `max-w-[240px]`, `h-8`, `text-sm`, outline variant, with Search icon (16x16) inline-start
- Faceted filter chips: `Button variant="outline" size="sm"`, gap of 8px between chips
- Action buttons: right-aligned, `Button variant="default" size="sm"` for primary CTA
- Toolbar gap: 8px between all items (`gap-2`)
- Toolbar padding: 0 (toolbar is flush inside DataTable container)
- Space below toolbar to table: 16px (`mb-4`)

### DataTable Pagination Bar

```
[Showing 1-10 of 234]  ----stretches----  [Rows per page: 10 v]  [< Prev] [1] [2] [3] [...] [Next >]
```

- Item count text: `text-sm text-muted-foreground`
- Page size selector: `Select` component, `w-[70px]`, `h-8`
- Page buttons: `Button variant="outline" size="icon-sm"` (28x28px), active page uses `variant="default"`
- Space above pagination from table: 16px (`mt-4`)
- Pagination bar alignment: `items-center justify-between`

### DataTable Row Actions

- Trigger: `Button variant="ghost" size="icon-sm"` (28x28px) with `MoreHorizontal` icon (16x16)
- DropdownMenu: `align="end"`, `w-[160px]`
- Menu items: `text-sm`, 32px height each
- Destructive items: `text-destructive` class, separated by `DropdownMenuSeparator`
- Separator before destructive items always

### Empty State

- Container: centered, `py-24` vertical padding
- Icon: 48x48, `text-muted-foreground/50` opacity
- Heading: `text-lg font-semibold`, 8px below icon
- Description: `text-sm text-muted-foreground`, 4px below heading, max-width 280px, centered
- CTA button (optional): `Button variant="outline" size="sm"`, 16px below description

### Loading Skeleton

- 5 rows of skeleton cells matching the column layout
- Each skeleton cell: `Skeleton` component, `h-4` height, varying widths (60%-80% of cell)
- Checkbox column skeleton: `Skeleton` at `h-4 w-4`
- Action column skeleton: `Skeleton` at `h-4 w-4` (circle)

## Date Formatting

All date display uses `date-fns` v4.1.0 (already installed).

| Context | Format | Example |
|---------|--------|---------|
| DatePicker button label | `PPP` (long date) | "April 17, 2026" |
| DateRangePicker button label | `MMM d` for both dates | "Apr 10 - Apr 17, 2026" |
| Table cell (date column) | `MMM d, yyyy` | "Apr 17, 2026" |
| Table cell (datetime column) | `MMM d, yyyy HH:mm` | "Apr 17, 2026 14:30" |
| Table cell (relative) | existing `formatRelativeTime` utility | "2 hours ago" |

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| Sort column keyboard | `button` role on column header, Enter/Space to toggle sort |
| Sort direction announcement | `aria-sort="ascending"` / `aria-sort="descending"` / `aria-sort="none"` on `<th>` |
| Row selection | `aria-selected="true"` on selected `<tr>`, checkbox has `aria-label="Select row"` |
| Select all | Header checkbox has `aria-label="Select all rows on this page"` |
| Pagination | `nav` with `aria-label="Pagination"`, current page button has `aria-current="page"` |
| DatePicker popover | Popover has `role="dialog"`, Calendar has built-in keyboard navigation from react-day-picker |
| Faceted filter | Popover trigger announces active filter count via `aria-label` |
| Empty state | Uses `role="status"` so screen readers announce when table has no content |
| Loading state | Skeleton area has `aria-busy="true"` and `aria-label="Loading data"` |
