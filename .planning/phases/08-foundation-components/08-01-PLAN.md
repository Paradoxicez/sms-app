**data-table-row-actions.tsx** (per D-03):

- Maps actions array to DropdownMenuItem elements; destructive items get `variant="destructive"` and are separated by DropdownMenuSeparator

**data-table-faceted-filter.tsx** (per D-05):
- Component `DataTableFacetedFilter<TData, TValue>` accepting `column: Column<TData, TValue>`, `title: string`, `options: Array<{ label: string; value: string; icon?: React.ComponentType<{ className?: string }> }>`
- Renders `Button variant="outline" size="sm"` as trigger, showing title and active count badge (e.g., "Status (2)")
- Opens Popover (from `@/components/ui/popover`) with PopoverContent

- Each option has a Checkbox + label. Checking toggles `column.setFilterValue()`
- Filter state uses `column.getFilterValue()` as `string[]`
- "Clear filters" button at bottom when any filters active
- Badge component shows count when `selectedValues.size > 0`

**data-table-toolbar.tsx** (per D-04, D-05, D-06):
- Component `DataTableToolbar<TData>` accepting `table: ReactTable<TData>`, `searchKey?: string`, `searchPlaceholder?: string`, `facetedFilters?: FacetedFilterConfig[]`, `children?: React.ReactNode` (slot for action buttons)
- Layout: flex row with `items-center gap-2`
- Left: Input component for search, `max-w-[240px] h-8`, with Search icon (16x16), filters rows using `table.getColumn(searchKey)?.setFilterValue(value)`

- Export `FacetedFilterConfig` type: `{ columnId: string; title: string; options: { label: string; value: string; icon?: React.ComponentType }[] }`

**data-table-pagination.tsx** (per D-07, D-08):
- Component `DataTablePagination<TData>` accepting `table: ReactTable<TData>`
- Layout: flex row `items-center justify-between mt-4`
- Left: `text-sm text-muted-foreground` showing "Showing {start}-{end} of {total}" computed from `table.getState().pagination` and `table.getRowCount()`

- Page number logic: show max 5 visible pages with "..." ellipsis. If pageCount <= 7, show all. Otherwise show first, last, current +/- 1, with "..." gaps.
- Selected row count: when `table.getSelectedRowModel().rows.length > 0`, show "{count} selected" text in muted-foreground

**data-table.tsx** (per D-01, D-08, D-12):
- Generic component `DataTable<TData, TValue>` with props:
  - `columns: ColumnDef<TData, TValue>[]` (required)
  - `data: TData[]` (required)
  - `searchKey?: string` -- column accessor key for text search
  - `searchPlaceholder?: string` -- defaults to "Search..."
  - `facetedFilters?: FacetedFilterConfig[]` -- config for filter chips
  - `toolbar?: React.ReactNode` -- slot for right-side action buttons
  - `enableRowSelection?: boolean` -- defaults to false
  - `onRowSelectionChange?: (rows: TData[]) => void` -- callback with selected rows
  - `pageCount?: number` -- for server-side pagination (enables manualPagination)
  - `onPaginationChange?: (pagination: { pageIndex: number; pageSize: number }) => void` -- server-side pagination callback
  - `loading?: boolean` -- shows skeleton state
  - `emptyState?: { icon?: React.ReactNode; title: string; description: string; action?: React.ReactNode }`
- Internal state: `sorting: SortingState`, `columnFilters: ColumnFiltersState`, `rowSelection: RowSelectionState`, `pagination: PaginationState` (default pageSize: 10)
- Calls `useReactTable` with `getCoreRowModel`, `getSortedRowModel`, `getFilteredRowModel`, `getPaginationRowModel` for client-side mode
- For server-side mode (when `pageCount` is provided): adds `manualPagination: true`, `manualSorting: true`, `manualFiltering: true`, `pageCount`
- Renders: `DataTableToolbar` (top) -> `Table` with `flexRender` for headers and cells -> `DataTablePagination` (bottom)
- Loading state: 5 skeleton rows using Skeleton component, matching column count
- Empty state: centered in table body with icon, title, description from `emptyState` prop; filtered-no-results shows "No results found" with "Clear filters" button
- Selected rows: `data-state="selected"` on TableRow when `row.getIsSelected()` is true
- Effect: when `rowSelection` changes and `onRowSelectionChange` provided, call it with `table.getSelectedRowModel().rows.map(r => r.original)`
