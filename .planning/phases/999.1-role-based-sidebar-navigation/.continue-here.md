## Critical Anti-Patterns (do NOT repeat these)

| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| hls_ctx changes m3u8 format | SRS returns master playlist redirect, not direct media playlist | blocking | Check for #EXT-X-STREAM-INF, follow inner URL |
| Caching SRS hls_ctx URLs | Sessions expire quickly, cached URLs go stale silently | blocking | Cache token only, resolve hls_ctx fresh |
| Middleware before guards | req.apiKey undefined in NestJS middleware | advisory | Defer guard-dependent checks to res.end |
