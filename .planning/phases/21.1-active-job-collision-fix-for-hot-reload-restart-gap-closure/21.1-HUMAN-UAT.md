## Reproducer — BKR06 + SD640 11-PATCH (mirror of 21-VALIDATION.md scenario)

**Precondition:** BKR06 must have a live FFmpeg process (verify in step D.3). SD640 (the stream profile used by BKR06) must exist.

**Commands assume super admin session cookie or API key — same auth used in Phase 21 manual UAT.**

| # | Test | Command | Expected (Phase 21.1) | Expected pre-fix (Phase 21) |
|---|------|---------|-----------------------|------------------------------|
| 1 | Capture pre-test FFmpeg PID | `ps aux \| grep "ffmpeg.*BKR06" \| grep -v grep` | record PID, e.g. `PID_BEFORE=14013` | (same) |
| 2 | Capture pre-test audit count | `psql ... -c "SELECT count(*) FROM \"AuditLog\" WHERE action = 'camera.profile_hot_reload' AND \"createdAt\" > now() - interval '1 minute';"` | `0` | `0` |
| 3 | PATCH #1 — change `SD640.videoBitrate: 2000k → 2500k` | `curl -X PATCH ... /api/stream-profiles/SD640 -d '{"videoBitrate":"2500k"}'` | `200`, `affectedCameras: 1`, `restartTriggered: true` | (same — surface always was correct) |
| 4 | PATCHes #2-11 — alternate `2500k ↔ 3000k` rapidly (10 more) | shell loop, 10 PATCHes ≤ 30s | each: `200`, `affectedCameras: 1` | (same) |
| 5 | Capture post-test audit count | (same as #2) | `11` | `11` |
| 6 | **Capture post-test FFmpeg PID** | `ps aux \| grep "ffmpeg.*BKR06" \| grep -v grep` | **PID changed (≥1 time, possibly up to 11 times)** ⭐ | **PID UNCHANGED — same as PID_BEFORE** ✗ |
| 7 | Verify final profile applied to live FFmpeg | `ps -o command= -p $(ps aux \| grep "ffmpeg.*BKR06" \| grep -v grep \| awk '{print $2}')` | command line shows last bitrate set (e.g. `-b:v 3000k` if PATCH #11 set 3000k) | command line shows `-b:v 2000k` (boot value) ✗ |
| 8 | Verify BullMQ active job state | `redis-cli HGETALL "bull:streams:camera:BKR06:ffmpeg"` | `data.profile.videoBitrate` matches PATCH #11 final value | `data.profile.videoBitrate: "2000k"` (stale boot value) ✗ |

**Pass criteria:** Step 6 + Step 7 are the load-bearing assertions. Audit rows (Step 5) were already correct in Phase 21. The fix is verified iff **FFmpeg PID changed at least once during the 11-PATCH window** AND **the live FFmpeg command line reflects the final PATCH'd profile**.
