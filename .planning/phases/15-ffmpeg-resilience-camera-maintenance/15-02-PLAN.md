| BullMQ Redis payloads → CameraHealthProcessor / BootRecoveryService | Payload trust boundary — Redis runs in private network; payloads built server-side. |
| Prisma cross-org queries in lifecycle services | No org context — by design, resilience runs cluster-wide. |
