## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP request body → controller/DTO | Untrusted JSON bodies from authenticated users. zod is the only sanitizer before the streamUrl touches FFmpeg, ffprobe, or SRS. |
| Bulk import CSV/Excel → row-by-row DTO | One authenticated user can submit up to 500 URLs in a single request. Each row must pass the same allowlist individually. |

Task 1/2/3 acceptance_criteria verify `javascript:`, `file://`, `http://` rejection. T-19-01 covered at Task 3 test cases explicitly. |

FFmpeg invocations wrap URL in double quotes (unchanged from existing code). See RESEARCH §"Security Domain → Known Threat Patterns". |
| T-19-01-b | Info Disclosure (zod error leaks) | BulkImportCameraSchema | LOW | accept | Zod error messages expose only the allowlist rule text, not the submitted URL — no PII/secret echo. |
