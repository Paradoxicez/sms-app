3. Create `settings.service.ts`:
- `getSystemSettings()` — findFirst from SystemSettings, create with defaults if none exists
- `updateSystemSettings(dto)` — update row, then call `this.regenerateAndReloadSrs()`
- `getOrgSettings(orgId)` — findUnique by orgId, create with defaults if none exists
- `updateOrgSettings(orgId, dto)` — upsert by orgId
- `generateSrsConfig(settings: SystemSettings): string` — produce srs.conf content using template literal with all settings values. Include HLS config (hls_fragment, hls_window, hls_use_fmp4 on, hls_cleanup on, hls_dispose 30, hls_wait_keyframe on). Include hls_keys block only if hlsEncryption is true. Include http_hooks for all 6 callbacks. Include rtc { enabled on; rtmp_to_rtc on; }.
- `regenerateAndReloadSrs()` — call generateSrsConfig(), write to `config/srs.conf` (or use SRS_CONFIG_PATH env), call srsApiService.reloadConfig()

The srs.conf template:
```
listen              ${settings.rtmpPort};
max_connections     1000;
daemon              off;
srs_log_tank        console;

http_server {
    enabled         on;
    listen          ${settings.httpPort};
}

http_api {
    enabled         on;
    listen          ${settings.apiPort};
}

stats {
    network         0;
}

vhost __defaultVhost__ {
    hls {
        enabled         on;
        hls_fragment    ${settings.hlsFragment};
        hls_window      ${settings.hlsWindow};
        hls_cleanup     on;
        hls_dispose     30;
        hls_wait_keyframe on;
        hls_use_fmp4    on;
${settings.hlsEncryption ? `        hls_keys        on;
        hls_fragments_per_key 10;
        hls_key_file    [app]/[stream]-[seq].key;
        hls_key_file_path /usr/local/srs/objs/nginx/html;
        hls_key_url     /keys/[app]/[stream]-[seq].key;` : ''}
    }

    http_hooks {
        enabled         on;
        on_publish      http://api:3001/api/srs/callbacks/on-publish;
        on_unpublish    http://api:3001/api/srs/callbacks/on-unpublish;
        on_play         http://api:3001/api/srs/callbacks/on-play;
        on_stop         http://api:3001/api/srs/callbacks/on-stop;
        on_hls          http://api:3001/api/srs/callbacks/on-hls;
        on_dvr          http://api:3001/api/srs/callbacks/on-dvr;
    }

    rtc {
        enabled     on;
        rtmp_to_rtc on;
    }
}
```
