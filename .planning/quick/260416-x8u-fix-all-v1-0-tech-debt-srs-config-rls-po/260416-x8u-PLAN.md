- `StreamProfile` -- column: `"orgId"`

**Policy naming convention:** `tenant_isolation_{table_lowercase}` and `superuser_bypass_{table_lowercase}` (matching existing pattern).

**Header comment:** `-- RLS Policies for Phase 02 Tenant-Scoped Tables`
