## Decisions Made

1. **Server `redirect()` over client `useRouter.replace()`** — proper 307 at request time, no `"use client"` bundle for an empty page, no `<Skeleton />` flicker, no useRouter dep. Drops two redundant client-side shims (the tenant Overview page and the admin Overview redirect twin both had `useEffect → router.replace(...)` patterns).
2. **Drop bare `/app/developer` from ROLE_MATRIX.developer** — listing a redirect-only path in the role matrix would be misleading; the destination `/app/developer/docs` is already permitted, and the redirect fires before any role-gate UI renders.
3. **Keep two-level breadcrumb in DocPage** — repoint the root link to `/app/developer/docs` rather than collapse to one crumb. Preserves visual hierarchy on deep pages like `/app/developer/docs/api-workflow` and matches today's UX (label "Developer" stays — only the `href` changes).
