| Category | Count | Examples |

| Linux-vs-Docker-Desktop divergence | 1 | srs.conf used `host.docker.internal` for callbacks — works on Mac, NXDOMAIN on Linux |

| Init-container exit semantics | 1 | bootstrap.sh `compose up -d --wait` for restart:"no" container → false-fail on exit 0 |
| IFS hardening side-effect | 1 | All 9 deploy scripts set `IFS=$'\n\t'` then used unquoted `${DC}` — bash treated whole var as one token |
| Caddyfile config | 2 | `acme_ca {$VAR:default}` parses set-but-empty wrong; missing `/live/*` route for SRS master→inner redirect |
| Frontend bundle baked-in dev URL | 1 | `NEXT_PUBLIC_API_URL || "http://localhost:3003"` baked into prod web bundle in 7 components |
| Stream Profile audio defaults | 1 | seed `audioCodec='aac'` failed on video-only RTSP cameras |
| Throttler pool sizing | 1 | NestJS Throttler 100/min global pool too tight for 7+ cameras polling |
| Security cleanup | 1 | 3 `console.log` calls leaking token + invitation URL + admin password |
